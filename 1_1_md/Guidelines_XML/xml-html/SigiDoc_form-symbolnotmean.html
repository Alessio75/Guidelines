2.3.4. Symbols (cross, leaf, scroll, fleurette, etc.)

Leiden symbols

  • Editorial interpretation: +, ((cross)), ((+))
  • Diplomatic edition: none

Explanation

Not meaning-bearing symbols, i.e. not to be expanded.

These symbols should be simply added by typing those supposed to appear in the editorial interpretation; for the diplomatic edition, Atherna Ruby should be used. Alternatively, the symbols can be tagged as follows in the editorial interpretation: 

                    <g ref="#cross"/> = ((cross))

or, if a charDecl is added to encodingDesc: 

                    <charDecl>
                        <glyph xml:id="cross">
                            <charProp>
                                <localName>glyph-display</localName>
                                <value>+</value>
                            </charProp>
                        </glyph>
                    </charDecl> = ((+))
