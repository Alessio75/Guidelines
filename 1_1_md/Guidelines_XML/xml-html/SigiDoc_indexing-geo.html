Place/geographical names

Elements

  • <placeName/>
  • <geogName/>

Explanation

1. Political entities (cities, peoples, regions, themes, etc.), but also buildings, etc.: 

                    <placeName type="" ref="">[name here]</placeName>

Because multiple tagging is possible, it could be used in different but complementary ways, ex. for the cities:

• to depict the typology of a place: type="city"

• to depict the function for which the name is used on the seal: type="bishopric"

Ethnonyms should be tagged as place names: e.g. in "bishop of the Nikopolitôn", "Nikopolitôn" should be indexed as Nikopolis.

2. Physical features (rivers, mountains, etc.):

                    <geogName type="" ref="">[name here]</geogName>

• Designed for the actual place + reference to the authority list.

• The type attribute could be used to specify what kind of place/feature is: city, theme, diocesis, church, etc., and it can also be used to identify ethnic names as opposed to the place names themselves.

• The spelling should be normalised in the authority list.
