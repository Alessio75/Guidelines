Miscellaneous

Elements

  • <rs/>

Explanation

Important: rs should be specified by a type attribute, whose value(s) listed here are the only being currently handled by SigiDoc/EFES autority lists and they should be used as they appear here.

Intended for religious, political, military, economical, etc. terminology / Remarkable expressions.

Marian terms:

                    <rs type="marianTerm" subtype="appellative" ref="marianTerms.xml#Δέσποινα">Δέσποινα</rs> (other ex.: Theotoke in the invocation, or vocative in the legend; different from label)
                    <rs type="marianTerm" subtype="label" ref="marianTerms.xml#..."/> (i.e. the name of the Virgin’s type inscribed on the seal [in the field], different from appellative)
                    <rs type="marianTerm" subtype="sigla" ref="marianTerms.xml#..."/> (ex.: ΜP ΘV)

Christ-related terms:

                    <rs type="christTerm" subtype="appellative" ref="christTerms.xml#..."/>
                    <rs type="christTerm" subtype="label" ref="christTerms.xml#..."/>
                    <rs type="christTerm" subtype="sigla" ref="christTerms.xml#..."/>

Saints-related terms:

                    <rs type="saintsTerm" subtype="appellative" ref="saintsTerms.xml#..."/> (Archistrategos for Michael, martyr, ...)
                    <rs type="saintsTerm" subtype="label" ref="saintsTerms.xml#..."/> (i.e. the name of the saint’s type inscribed on the seal [in the field], different from saint's name; the adjective hagios should be included in the tag)
                    <rs type="saintsTerm" subtype="sigla" ref="saintsTerms.xml#..."/> (ex.: Μ Χ)
                    <rs type="saintsTerm" subtype="name" ref="saintsTerms.xml#..."/> (for ex., in an invocation in the legend; different from saint's label)

Important: the attribute subtype is paramount in order to show in the Indices the difference among the various types of terms.

With the subtype 'appellative' should also be tagged the adressees of the invocations, the full text of which are however contained in a separate index (see "Invocations").

Other examples:

                    <rs type="ecclesiastical" ref="">ποιμήν</rs>
                    <rs type="devotional" ref="">δοῦλος τῆς Θεοτόκου</rs> (connected to the seal's owner)
                    <rs type="divinity" ref="">Κύριε</rs> (Trinity, different names for Christ, Theos, Cross)
                    <rs type="genealogical" ref=""/> (only for explicit genealogical references)

And so on... The choice of what kind of miscellaneous terms to index is a project’s decision.

The spelling should be normalised in the authority list.
