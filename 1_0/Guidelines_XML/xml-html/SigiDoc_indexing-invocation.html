Invocations

Elements

  • <rs/>

Explanation

Important: rs should be specified by a type attribute, whose value(s) listed here are the only being currently handled by SigiDoc/EFES autority lists and they should be used as they appear here: 

                    <rs type="invocation" ref="">[invocation here]</rs>

To rs type="invocation" a rend attribute could be added to give information about the physical presentation of the inscription: 

                    <rs type="invocation" ref="" rend="linear"/>
                    <rs type="invocation" ref="" rend="circular"/>
                    <rs type="invocation" ref="" rend="monogram"/>
                    <rs type="invocation" ref="" rend="column"/>
                    ...
