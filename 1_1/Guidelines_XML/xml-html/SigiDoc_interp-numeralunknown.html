Numerals: value unknown because lost or illegible

Leiden symbols

  • Editorial interpretation: 1) [.. ? ..], 2) .. ? ..
  • Diplomatic edition: none

Explanation

                    1) <num>
                        <gap reason="lost"/>
                       </num>
                    
                    2) <num>
                        <gap reason="illegible"/>
                       </num>

It is visualised as a gap.

It is possible to add attributes to gap.
