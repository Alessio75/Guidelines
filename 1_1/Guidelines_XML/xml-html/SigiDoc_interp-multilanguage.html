Leiden symbols

  • Editorial interpretation: abc αβγ
  • Diplomatic edition: abc αβγ

Explanation

                    <div type="edition" subtype="editorial/diplomatic" xml:space="preserve" xml:lang="grc-Latn"/>

In case of bilingual Greek/Latin legends, an xml:lang with value 'grc-Latn' should be added to the div type="edition" containing all the legend.

Note that this value is used here not only for different languages, but also for different alphabets, i.e. for a Greek legend written in Latin alphabet.
