4. List of All Indexing Guidelines (currently being reassessed)

In this index is displayed the set of rules designed for SigiDoc in order to extract (potentially) all the information coming from a seal's legend and organise it in indices automatically generated through EFES. The following thematic list contains the most relevant topics, unavoidable in sigillographic indices and traditionally appearing in most of the sigillographic corpora: however, the ultimate choice about the themes represented in indices belongs to each projet, expanding or shortening this list, but, especially in the latter case, it is strongly recommended to keep a common range of indices among the projects using SigiDoc, in order to ensure interoperability and searchability.

This represents the first step in creating and populating indices: for complete guidance, please, follow the EFES guidelines.
